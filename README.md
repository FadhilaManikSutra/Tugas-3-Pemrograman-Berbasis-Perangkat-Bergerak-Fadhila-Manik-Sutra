# Crypto Market Dashboard

## Overview

Crypto Market Dashboard is a mobile-friendly web application developed using **Ionic Framework** and **Vue.js**. The application retrieves real-time cryptocurrency data from the CoinLore API and presents it through a clean and responsive user interface.

This project was developed as part of the Mobile-Based Programming practical assignment.

---

## Features

- Display Top 10 Cryptocurrencies
- Real-Time Data from CoinLore API
- Cryptocurrency Search Function
- Dark Mode / Light Mode Toggle
- API Connection Status Indicator
- Refresh Data Button
- Responsive Design
- Modern and Minimalist User Interface

---

## Technologies Used

- Ionic Framework
- Vue.js 3
- TypeScript
- Axios
- CoinLore API

---

## API Source

Data is retrieved from:

https://api.coinlore.net/api/tickers/

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/crypto-market.git
```

Navigate to the project folder:

```bash
cd crypto-market
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
ionic serve
```
